# Daten_Beschaffung

Dieser Ordner enthält das Notebook zur **automatisierten Beschaffung der GDELT-Daten**.

## Inhalt

- Download der GDELT-Eventdaten für das Jahr 2019
- Speicherung der Rohdaten im ZIP-Format
- Entpacken der ZIP-Dateien in ein lokales Rohdatenverzeichnis

Die Rohdaten werden **nicht** in das Repository eingecheckt, sondern lokal erzeugt.

## Zweck

Dieses Notebook stellt sicher, dass die im Analyse-Notebook verwendeten Daten
reproduzierbar aus der Originalquelle bezogen werden können.
