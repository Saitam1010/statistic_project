# Explorativ

Dieser Ordner enthält **explorative Notebooks und Zwischenschritte**, die während
der Projektentwicklung entstanden sind.

## Zweck

- Erste Dateninspektion
- Testen von Variablenableitungen
- Methodische Vorversuche

## Hinweis

Die Inhalte dieses Ordners sind **nicht Teil der finalen Abgabe** und dienen
ausschliesslich der Nachvollziehbarkeit des Entwicklungsprozesses.
